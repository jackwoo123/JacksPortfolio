This Summer (2021) in addition to Smuckers, I participated in the EA Virtual Internship provided by Forage. As apart of this virtual internship, I created a verision of Pac Man but modified it by changing the rules to a game called Vax Man.



Vax Man was pretty similar to Pac Man. But there are some rule changes, the main difference is that instead of the ghosts killing Pac Man upon contact, Pac Man actually kills the ghosts or as EA said, "vacinates them". Every 30 seconds the number of ghosts on screen would double, requiring Vax Man to quickly catch all the ghosts.





Challenges Encountered


The main challenge I have encountered so far was learning to read others code. I think this was the most important challenge to overcome because of the industry norm of reading and working with others peoples code. I overcame this challenge by before starting coding by documenting the code way before I ever started coding.
