Software Two


Software Two focused on data structures further like sets, stacks, queues, and more maps. We learned about kernal interfaces which was really interesting to me. Kernal interfaces use the same interfaces, but use different code to achieve the method contracts. This was the main topic because we had alot of labs that implemented a map, using a set for example.
